## Session 7 — Git Branches and Documentation

In this session, I practiced cloning a repository, creating branches, switching between branches, using .gitignore, committing changes, merging branches, and improving DevOps documentation. I learned that branches allow engineers to work on changes safely before merging them into the main codebase.
